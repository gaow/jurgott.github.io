# ALLEGRO
Record added by Jurg Ott (the original Rockefeller list)

## Version
2.0 (October 2005)

## Description
a faster version of GENEHUNTER and Allegro 1.2 (several degrees of increase of speed, can handle bigger families, up to 50 bits)

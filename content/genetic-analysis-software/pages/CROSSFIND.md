# CROSSFIND
Record added by Jurg Ott (the original Rockefeller list)

## Other Name
s: BREAKPOINT, BPT

## Version
October 1995

## Description
software for detecting and displaying well-characterized meiotic breakpoints in human family data. another program, CRIMAP, is used.
